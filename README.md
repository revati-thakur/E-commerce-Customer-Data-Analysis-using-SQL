# E-commerce-Customer-Data-Analysis-using-SQL
This project focused on cleaning and analyzing customer data from an E-commerce dataset using SQL. The project included data validation, null value checks, duplicate detection, data cleaning, and exploratory data analysis (EDA) to understand customer demographics and behavior patterns.

Performed data cleaning, validation, and exploratory data analysis (EDA) on an E-commerce customer dataset using SQL. The project focused on improving data quality, identifying duplicates, handling missing values, standardizing customer information, and generating business insights through customer segmentation.

## Key Tasks Performed
- Data overview and schema analysis
- Null value detection and handling
- Duplicate customer identification using ROW_NUMBER()
- Data standardization and cleaning
- Gender value correction and formatting
- Customer age validation and imputation
- Customer segmentation based on tenure
- Top customer analysis
- Clean dataset creation

## SQL Concepts Used
- CTEs
- Window Functions (ROW_NUMBER)
- CASE Statements
- Aggregate Functions
- Data Cleaning
- Joins & Filtering
- Group By & Order By

## Business Insights
- Customer age distribution analysis
- Gender-based customer analysis
- Tenure-based customer segmentation
- Identification of high-tenure customers
